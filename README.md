Данный проект является CLI для генерации ER-диаграммы в формате UML.

В основе формирования UML файла используется синтаксис PlantUml.

Для генерации UML файла необходимо в директории проекта ввести в терминале команду "dotnet run (необязательный аргумент для названия файла) --path (путь к сборке проекта .dll) --save (путь сохранения файла .puml)"

Для проверки формирования UML можно использовать данный сайт: https://www.planttext.com
Предварительно скопировав содержимое из файла в формате .puml
